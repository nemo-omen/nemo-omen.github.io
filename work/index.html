Mando

Mando is an OBS remote that was created to fill a specific need within a digital streaming studio — the need to direct a show from the other side if the studio.

To that end, Mando was created as a functional recreation of the OBS Studio interface, including basic scene-selecting and scene-switching, and on-screen previews.

Mando was built with SvelteKit.

Prompter

Prompter is another tool built specifically for digital broadcasting. It is intended for use in locations where internet connectivity is primarily reserved for streaming output.

For that reason, Prompter persists all of its data to the browser's LocalStorage, bypassing the need for an internet connection and heavier, complicated database management systems.

Prompter was built with Svelte.

TrainingMontage.dev

TrainingMontage.dev is my personal blog, where I write about various technical subjects as a way to solidify my own learning while sharing what I've learned with anyone who might be new to programming.

Use of web-platform fundamentals to achieve a responsive, accessible design were top priorities while building this site and the conscious decision was made to include color-schemes that are controlled by device-level user preferences.

The site was built using Astro, a new static-site-generator that helps to ensure as little JavaScript as possible is served to the browser.
